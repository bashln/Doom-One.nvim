# Doom-One.nvim

Tema inspirado no Doom One do Doom Emacs, convertido para Neovim.

## Instalação

```lua
-- lazy.nvim
{ "seu-usuario/Doom-One.nvim" }
```

## Uso

```lua
require("doom-one").setup({
  transparent = false,
})

vim.cmd.colorscheme("doom-one")
```
